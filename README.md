# Portfólio — Dev Full Stack

Este é meu portfólio pessoal como desenvolvedor Full Stack, construído com [Next.js](https://nextjs.org) e inicializado com [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## 🚀 Como rodar o projeto

Instale as dependências:

```bash
pnpm install
```

Rode o servidor de desenvolvimento:

```bash
pnpm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no navegador para ver o resultado.

> Este projeto utiliza [pnpm](https://pnpm.io) como gerenciador de pacotes.

## 🛠️ Tecnologias

- [Next.js](https://nextjs.org) — framework React para produção
- [TypeScript](https://www.typescriptlang.org) — tipagem estática
- [Tailwind CSS](https://tailwindcss.com) — estilização
- [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) — otimização de fontes
